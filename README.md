# ⏳ FocusFlow Pro

FocusFlow Pro is a modern web-based productivity timer designed to help users stay focused, manage time effectively, and track their study/work sessions. It combines a clean UI with powerful features like focus mode, session tracking, and visual analytics.

---

## 🚀 Live Demo

👉 https://tharunkorla.github.io/Focus-Flow/

---

## ✨ Features

* ⏱️ **Custom Timer** (Hours, Minutes, Seconds)
* 🔔 **Loud Alarm** with auto-stop (1 minute)
* 🎯 **Focus Mode** for distraction-free work
* 📊 **Productivity Chart** (visual session tracking)
* 📈 **Progress Bar** (real-time progress)
* ⚡ **Preset Timers** (25m, 1h, etc.)
* 🧠 **Session Tracking** (study time calculation)
* 🎨 **Modern UI** (glass effect, animations, gradients)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling (Glass UI, animations)
* **JavaScript (Vanilla JS)** – Logic & functionality
* **Chart.js** – Data visualization

## 💡 How to Use

1. Enter your name on the welcome screen
2. Set a timer or choose a preset
3. Click **Start** to begin your session
4. Track your progress with the progress bar
5. Use **Focus Mode** for deep work
6. View your session data in charts
